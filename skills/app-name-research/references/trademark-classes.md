# USPTO Trademark Classes for Apps

## Primary Classes for Mobile Apps

### Class 9 - Computer Software
**Use for:** Mobile apps, downloadable software, computer programs

Includes:
- Mobile applications (iOS, Android)
- Downloadable software
- Application software
- Computer game software
- Educational software

**Example goods:**
- "Downloadable mobile application for [purpose]"
- "Computer application software for [purpose]"

### Class 42 - Software as a Service (SaaS)
**Use for:** Web-based applications, cloud services, platforms

Includes:
- Software as a service (SaaS)
- Platform as a service (PaaS)
- Application service provider (ASP) services
- Cloud computing services

**Example services:**
- "Software as a service (SaaS) services featuring software for [purpose]"
- "Platform as a service (PaaS) featuring software for [purpose]"

## Secondary Classes (Based on App Purpose)

### Class 35 - Business Services
**Use for:** Apps offering business services, marketing, advertising

- Business management apps
- Marketing platforms
- Customer relationship management (CRM)
- E-commerce platforms

### Class 36 - Financial Services
**Use for:** Finance, banking, payment apps

- Banking apps
- Payment processing
- Investment platforms
- Cryptocurrency apps

### Class 38 - Communication Services
**Use for:** Messaging, social, communication apps

- Messaging apps
- Social networking platforms
- Video calling services
- Email services

### Class 41 - Education and Entertainment
**Use for:** Educational apps, games, media streaming

- Educational apps
- Gaming apps
- Streaming platforms
- Online courses

### Class 44 - Medical/Health Services
**Use for:** Health apps, telemedicine, wellness apps

- Health monitoring apps
- Telemedicine platforms
- Wellness tracking
- Medical appointment scheduling

### Class 45 - Personal Services
**Use for:** Personal safety, lifestyle, family apps

- Safety monitoring apps
- Family tracking services
- Personal security services

## Filing Strategy

### Minimum Filing
- **Class 9** only: ~$250-350
- Protects the app software itself

### Recommended Filing
- **Class 9 + Class 42**: ~$500-700
- Protects both downloadable app AND web service

### Comprehensive Filing
- **Class 9 + Class 42 + Purpose Class**: ~$750-1050
- Example: Health app = Class 9 + Class 42 + Class 44
- Maximum protection

## Filing Options

### TEAS Plus ($250/class)
- Must use pre-approved descriptions
- Limited customization
- Lower cost

### TEAS Standard ($350/class)
- Custom descriptions allowed
- More flexibility
- Higher cost

## Trademark Strength Spectrum

From strongest to weakest:

1. **Fanciful** - Made-up words (Kodak, Exxon)
2. **Arbitrary** - Real words unrelated to product (Apple for computers)
3. **Suggestive** - Hints at function (Netflix)
4. **Descriptive** - Describes function (slowly gaining protection)
5. **Generic** - Common name for product (cannot be trademarked)

**Recommendation:** Aim for Fanciful or Arbitrary names for strongest protection.
